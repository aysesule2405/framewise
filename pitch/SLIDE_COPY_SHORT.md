# Framewise Short Pitch Deck

Use this if the presentation needs to be 6-8 slides.

## 1. Framewise

**Turn video into active learning.**

AI-powered video learning assistant for YouTube, tutorials, lectures, and practice videos.

## 2. Problem

**Video is powerful, but hard to study.**

- Key moments are buried in long timelines.
- Search inside videos is weak.
- Notes, captions, bookmarks, and chat are disconnected.
- YouTube does not remember the full learning workflow.
- Practice videos need looping, slowdown, and timestamped review.

## 3. Solution

**Framewise makes video behave like an interactive document.**

- AI topic timeline
- Timestamp-aware chat
- Bookmarks and notes
- Captions and translation
- Quizzes
- Collections
- Continue watching
- Practice mode

## 4. Two Connected Surfaces

**Web app + Chrome extension, one shared backend.**

Web app:
- Analyze videos
- Manage the library
- Search and organize learning

Chrome extension:
- Works directly on YouTube
- Auto-loads saved timelines
- Injects timed captions
- Syncs progress

## 5. Demo

**One video becomes a study workspace.**

Paste URL -> analyze -> click timeline -> chat -> bookmark -> generate captions/quiz -> open on YouTube -> extension auto-loads.

## 6. Technology

**Sponsor tools are product-critical.**

- Gemini: video understanding, segments, chat, quizzes
- MongoDB Atlas: videos, segments, captions, bookmarks, collections, progress
- ElevenLabs: voice replies and future hands-free coaching
- React, Express, Manifest V3, Google OAuth/JWT

## 7. What We Built

**MVP-plus, across frontend, backend, and extension.**

- Auth and shared API
- Video analysis and timeline
- Chat and voice
- Search, bookmarks, collections
- Continue watching
- Captions synced to video timing
- Extension auto-load
- Dark/light design system

## 8. Roadmap

**From hackathon build to launchable product.**

- Stabilize AI jobs and error states
- Add async queue for long tasks
- Support uploaded videos
- Improve caption editor/export
- Add study packs and spaced review
- Deploy web app/backend
- Prepare Chrome Web Store release
